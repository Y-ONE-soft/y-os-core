# 118. fix: 내 할일 요청 알림을 종류별 2열 배치로 변경

## 작업 요약

docs/116에서 요청 알림을 종류별로 나눌 때 **가로 줄(행)** 로 쌓았는데, 의도는 **세로 열**이었다. `할일 요청`과 `도움 요청`을 나란한 두 열로 바꾼다.

```
요청 알림 · 3        전체 보기

할일 요청 · 2          도움 요청 · 1
[카드]                 [카드]
[카드]
```

## 변경 내용

### `src/components/features/my-work/my-work-requests.tsx`

- `KIND_ROWS` → `KIND_COLUMNS`. 바깥을 `flex items-start gap-3`으로 두어 두 열이 가로로 나란히 서고, 각 열은 `flex-1`로 폭을 반씩 나눈다
- 카드는 열 안에서 **세로로 쌓인다**(`flex flex-col gap-2`). 이전의 가로 스크롤은 없앴다 — 열 안에서는 아래로 늘어나는 편이 자연스럽다
- 카드에서 `className="flex-1"`을 뺐다. 가로 나열일 때 폭을 나누려던 것인데 세로 스택에서는 필요 없다
- **비어 있어도 열은 남긴다.** 한쪽이 비었다고 다른 쪽이 전체 폭을 차지하면 건수에 따라 레이아웃이 출렁인다. 빈 열에는 점선 테두리로 `받은 OO 요청이 없습니다`를 둔다

docs/116에서는 빈 종류의 줄을 아예 숨겼는데, 2열 배치에서는 반대로 남기는 쪽이 맞다. 줄은 없어져도 위아래로만 짧아지지만, 열은 없어지면 옆 열의 폭이 바뀐다.

## 적용 범위

**내 할일 화면에만** 적용한다. 알림 페이지(`/notifications`)는 전체 목록·방향 필터를 담당하는 별도 화면이라 그대로 둔다. 이 파일(`my-work-requests.tsx`)은 내 할일 전용이라 자연히 범위가 맞는다.
