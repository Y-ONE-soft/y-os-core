# 56. 캘린더 단계 막대 드래그 이동·기간 조절

- **예정 커밋 메시지**: `feat: 캘린더 단계 막대 드래그 이동·기간 조절`
- **작업일**: 2026-07-22
- **작업 브랜치**: `캘린더-드래그` (워크트리 `.claude/worktrees/calendar-drag`, base: main `36d6641`)
- **요청 사이클 태스크 2/4** — 태스크 1(docs/55) 할일 칩 표시에 이어, 단계 막대를 드래그로 조작한다. 다음은 할일 드래그(3), 프로젝트 기간 필드·박스 드래그(4).

---

## 1. 작업 요약

내 작업 캘린더의 단계 막대를 **드래그로 이동**하고 **양 끝 손잡이로 기간을 조절**할 수 있다. 요청의 핵심 규칙인 "**항상 하위 항목을 커버하도록 늘어난다**"를 단계↔할일 관계에 적용했다.

## 2. 동작 규칙

| 조작 | 결과 |
|---|---|
| 막대 본체 드래그 | 시작·종료를 함께 이동. 이동 후에도 자기 할일을 덮어야 하므로, 벗어나는 쪽 끝이 **할일 날짜까지 늘어난다** |
| 왼쪽 손잡이 | 시작일 조절. 할일보다 늦게 밀 수 없다 — **가장 이른 할일 날짜에서 멈춘다** |
| 오른쪽 손잡이 | 종료일 조절. 할일보다 이르게 당길 수 없다 — **가장 늦은 할일 날짜에서 멈춘다** |
| 3px 미만 이동 | 드래그가 아니라 **클릭** — 기존대로 단계 상세 오버레이가 열린다 |
| 저장 시점 | 드래그 중에는 화면만 미리 움직이고, **손을 뗄 때 한 번** 저장한다 |

주 경계를 넘어 여러 조각으로 잘린 단계도 조각이 **한꺼번에** 따라온다 — 미리보기를 소스 단계에서 갈아끼우기 때문이다.

## 4. 검증

**브라우저 시나리오** (puppeteer, dev **3111** — 포트 소유권 확인 후 진행). 기준: 단계 `07-20~07-28`, 그 안의 할일 `07-26`.

| # | 조작 | 결과 |
|---|---|---|
| 1 | 끝 손잡이 ←4칸 | `07-20~07-26` — 할일 날짜에서 멈춤 ✓ |
| 2 | 시작 손잡이 →6칸 | `07-26~07-28` — 할일 날짜에서 멈춤 ✓ |
| 3 | 본체 ←3칸 | `07-17~07-26` — 이동하되 끝이 할일까지 늘어남 ✓ |
| 4 | 막대 클릭 | 단계 상세 오버레이 열림 ✓ |

- `npm run lint` ✓ · `npm run build` ✓ · **콘솔·네트워크 에러 0건**
- 같은 시나리오를 **2회 연속 실행해 동일 결과** 확인 (아래 재현성 이슈 때문에 필수였다)

### 검증에서 잡아낸 실제 버그 2건

1. **`clampStageToTasks`가 클램프 결과를 버렸다** — `endDate: dates.endDate ?? …` 형태라 종료일이 있는(대부분의) 단계에서는 항상 원래 값이 돌아왔다. 커버 규칙이 사실상 동작하지 않던 것을 테스트가 잡았다.
2. **드래그 중 조각이 언마운트되면 포인터 캡처가 끊겼다** — 막대 조각에 캡처를 걸었는데, 드래그로 날짜가 바뀌며 조각이 주 경계를 넘거나 사라지면 그 DOM이 사라져 이후 move·up이 오지 않았다(= 저장 안 됨). 처음엔 테스트 불안정으로 보였으나 **실패 조합이 매번 달라지는 양상**이 단서였다. 캡처를 **캘린더 루트**(안정적인 조상)로 옮겨 해결했고, 오버레이 키도 인덱스 → `id:col`로 바꿔 재배치 시 엉키지 않게 했다.

> 캡처를 pointerdown 즉시 걸었더니 이번엔 **막대 클릭이 먹지 않았다** — click이 캡처 대상(루트)으로 배달되기 때문. 임계값(3px)을 넘어 드래그로 확정된 뒤에만 캡처하도록 바꿔 클릭과 드래그를 모두 살렸다.

## 5. 알려진 사항 / 후속 과제

- 할일 칩 드래그는 태스크 3, 프로젝트 박스 드래그는 태스크 4
- 단계를 옮겨도 그 안의 **할일 예정일은 따라오지 않는다** — 커버 규칙상 단계가 늘어나 할일을 품는다. 할일을 함께 끌고 가는 방식이 필요하면 별도 논의
- 드래그는 하루 단위(칸 단위)로만 움직인다
- 프로젝트 박스는 아직 단계들의 합집합이라 단계를 옮기면 박스도 자동으로 따라온다
