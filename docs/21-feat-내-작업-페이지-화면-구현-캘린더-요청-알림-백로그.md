## 4. 결정 사항

- **변수 모드 raw 불일치 처리**: raw 코드의 캘린더 탭·기간 세그먼트·오늘 칸 `bg-primary`는 렌더 화면에선 연회색 — docs/4 선례대로 스크린샷 기준 `bg-accent` 채택. 반면 "작업 할당 요청" 칩·수락 버튼은 렌더도 진한 색이라 `bg-primary`/shadcn Button default 사용
- **오버레이 좌표**: 디자인의 절대 px 대신 주(week) 행 기준 `col/7 %` + 레인(20px) 시스템으로 재구성 — 반응형에서 칸에 정렬 유지
- **원형 번호·체크 사각형**: 단색 도형이라 에셋 다운로드 없이 CSS로 구현 (아이콘 글리프 아님)
- 스팬·요청 데이터는 `my-work-data.ts` 자리표시 상수 — 일정/요청 도메인 DB 도입 시 API 경계 흐름으로 교체
