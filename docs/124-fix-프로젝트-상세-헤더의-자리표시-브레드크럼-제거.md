# 124. fix: 프로젝트 상세 헤더의 자리표시 브레드크럼 제거

## 작업 요약

프로젝트 상세 헤더 맨 위의 브레드크럼(`{프로젝트명} › 고객 Y.ONE 내부`)을 제거했다. 자리표시라 실데이터가 없고 바로 아래 제목과 겹쳤다.

## 문제

```
Y.OS contents › 고객 Y.ONE 내부      ← 제거 대상
● Y.OS contents                      ← 제목 (그대로)
진행률 0%
```

두 가지가 겹쳤다.

1. **`고객 Y.ONE 내부`가 하드코딩** — 어느 프로젝트를 열든 항상 같은 문구가 나왔다. 실제 고객·소속 데이터가 아니다.
2. **앞 칩이 제목과 같은 프로젝트명을 중복** — 칩 `{project.name}`과 바로 아래 제목 `{project.name}`이 같은 값이라 프로젝트 이름이 두 번 보였다.

실데이터도 없고 아래 제목과 겹치는 줄이라 통째로 지웠다.

## 변경 내용

### `src/components/features/projects/project-detail-page.tsx`

헤더 제목 블록에서 브레드크럼 `div`(칩 + `›` + `고객 Y.ONE 내부`)를 삭제했다. 색 점 + 프로젝트 이름 제목과 진행률 요약줄은 그대로 둔다 — 프로젝트 식별에는 이걸로 충분하다.

## 왜 그룹명으로 바꾸지 않았나

브레드크럼을 실데이터(예: `Soft › Y.OS contents`)로 채우는 방법도 있었지만, 사용자 지시가 "필요없으면 지워줘"였고 아래 제목이 이미 프로젝트를 식별하므로 제거를 택했다. 계층 표시가 필요해지면 별도 사이클에서 그룹 관계를 붙인다.
