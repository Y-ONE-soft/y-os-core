## 1. 작업 내용 요약

작업 현황 페이지 헤더 우측의 **「＋ 작업」 버튼을 제거**했다 (사용자 요청).

- 이 버튼은 디자인(Task Status Layout)에는 있었지만 동작이 정의된 적 없는 **시각 자리표시**였다 (docs/14의 후속 과제로 남아 있던 항목).
- 작업 추가는 이미 실제 경로가 있다 — 프로젝트 상세 보드의 `＋ 작업`, 백로그 인라인 입력, 내 작업 페이지. 헤더 버튼은 동작 없이 오해만 유발했다.
