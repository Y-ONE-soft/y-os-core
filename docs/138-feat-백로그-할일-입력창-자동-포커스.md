# feat: 백로그 할일 입력창 자동 포커스

## 1. 작업 내용 요약

백로그 화면에 들어오면 **할일 추가 입력창에 커서가 자동으로 놓이도록** 했다. 입력창을 클릭할 필요 없이 바로 키보드로 쳐서 Enter만 누르면 백로그에 추가된다.

리뷰 피드백: "해당 화면에 들어가서 키보드 입력하면 자동으로 거기 써지고 엔터하면 반영되도록." 입력창은 이미 상시 보이고 Enter로 추가되는 형태였고(직접 입력), 여기에 **자동 포커스**만 더해 클릭 단계를 없앴다.

## 3. 구현

두 백로그(프로젝트 상세 우측 패널, 내 할일 우측 패널)의 add 입력에 `autoFocus`를 붙였다.

- 화면(패널)이 마운트될 때 입력이 포커스된다 → 들어오자마자 타이핑하면 그 값이 입력창에 들어간다.
- Enter 처리·추가 후 입력 비우기는 기존 그대로(직접 입력 방식). 그래서 연속 입력도 유지된다.
- 두 백로그는 서로 다른 페이지라 동시에 마운트되지 않아 포커스 경쟁이 없다.

## 5. 알려진 사항 / 후속 과제

- **자동 포커스는 마운트 시 1회다.** 화면에 들어온 뒤 백로그 항목을 클릭해 상세를 열었다 닫거나 다른 곳을 클릭하면 포커스가 벗어난다 — 그때는 다시 입력창을 클릭해야 한다. "언제든 아무 데서나 타이핑하면 입력창으로 캡처"(문서 전역 키 리스너)까지 원하면 별도 사이클이 필요하다.
- 프로젝트 상세는 주 콘텐츠가 보드(좌측)인데 입장 시 포커스가 우측 백로그로 간다. 사용자가 명시적으로 요청한 동작이다. 입력이 패널 상단이라 스크롤 점프는 없다.
