# feat: 할일 상세 세부사항에서 작업자 배정 항목 제거

## 작업 요약

할일 상세 오버레이 오른쪽 "세부 사항" 패널의 **"작업자 배정" 항목(담당자 지정 Select)** 하나만 제거했다.

- 남긴 것: 세부 사항 첫 항목인 "작업자"(현재 담당·공동 작업자 **표시**), 프로젝트·단계 Select, 예정일·완료날짜·유형·난이도 등 나머지 항목, 그리고 하단의 도움 요청·공동 작업자 지정 요청 버튼은 **그대로**.
- 제거한 것: `assigneeId`를 고르는 "작업자 배정" Select 블록.

## 변경 파일 내역

### `src/components/features/projects/task-detail-overlay.tsx`

- "작업자 배정" 라벨 + 담당자 지정 `Select`(미배정/사용자 목록) 블록 삭제 (약 43줄).
- 그 블록에서만 쓰던 상수 `NO_ASSIGNEE_VALUE` 정의 삭제 (미사용 → 린트 오류 방지).
- `users`·`usersLoading`·`avatarColor`는 하단 요청 다이얼로그에서 계속 쓰이므로 그대로 둠.

## 알려진 영향

- 이 Select는 할일의 `assigneeId`를 바꾸는 **한 진입점**이었다. 담당자 값 자체는 여전히 유효하며(작업 현황의 담당자 보드가 이 값으로 컬럼을 묶는다), 담당자 변경은 그 보드 등 다른 경로로 한다. 세부 사항의 "작업자" 표시는 남아 현재 담당자를 계속 보여준다.
