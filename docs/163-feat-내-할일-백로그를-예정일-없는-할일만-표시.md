# feat: 내 할일 백로그를 예정일 없는 할일만 표시

## 요청 (사이클 A 태스크 2)

> 백로그는 날짜도·단계도 없는 것. (예정일이 잡힌 "단계 없음" 할일은 캘린더에 뜬다.)

## 변경 — `src/components/features/my-work/my-work-backlog.tsx`

내 할일 페이지 좌측 "백로그" 패널은 미배정 할일 + 전 프로젝트 백로그를 모아 보여준다. 여기서 **예정일이 있는(=단계 없음) 할일을 제외**하고, **예정일 없는(백로그) 할일만** 남긴다.

- `items` 필터에 `&& !task.scheduledDate` 추가(담당자=나 조건은 그대로).
- 상단 주석에 "백로그 = 예정일·단계 둘 다 없음, 예정일 잡힌 것은 캘린더(단계 없음)로" 명시.

## 동작 흐름 (사이클 A 완성)

- **백로그(미정)**: 내 할일 백로그 패널 + 프로젝트 상세 "백로그" 섹션.
- **단계 없음(예정일 있음, 단계 없음)**: 캘린더(프로젝트 박스 / "프로젝트 없음" 그룹) + 프로젝트 상세 "단계 없음" 섹션(태스크 1).
- 왕복: 백로그 할일에 날짜를 잡으면(캘린더로 드래그·날짜 확정) 단계 없음이 되어 캘린더에 뜨고, 캘린더 칩을 백로그 패널로 드래그하면(#143, 예정일 해제) 다시 백로그로.

## 데이터/동작 영향

- **스키마·서버 변경 없음.** 표시 필터만.
- 기존에 이 패널에 뜨던 "예정일 있는 미배정/백로그" 할일은 이제 캘린더에서만 보인다(중복 표시 제거 효과).
